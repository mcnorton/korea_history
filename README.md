# 대한민국 역사연표

1. 항상 가지고 다니면서 역사를 공부
2. 수업시간 태블릿에 무료로 설치하여 사용
3. 한국사와 주요 세계사를 동시에 비교
4. 위키피디아를 통해 자세히 읽기
5. 나의 역사를 기록해서 연표 만들기
